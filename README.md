# odin-hangman
An ASCII art illustrated, terminal implementation of [Hangman](https://www.theodinproject.com/lessons/ruby-hangman), written in Ruby, for [The Odin Project](https://www.theodinproject.com/). 

Halfway through the Ruby track, this project has taught or reinforced for me:

- Basics of file reading and writing, line-by-line and all at once.
- Persisting app data via JSON file serialization and deserialization.
- Exception handling for file loading, parsing, and saving issues.
- Resolution of problematic Rubocopss on a file-specific basis.
- Omitting app-generated data via a custom gitignore file.

## Pre-requisites for Usage

Ruby 2.7+

## Instructions for Usage

Type `ruby main.rb` into your terminal to start playing the game.

## Rubocop Instructions

Type bundle exec rubocop -a into your terminal to detect and auto-resolve many issues.
